Stardew Valley Decay Mod

[English](readme.md) | [中文版本](readme_ZH.md)

This is a realism-enhancing mod for Stardew Valley that introduces a dynamic item shelf-life system along with a waste-to-treasure recycling mechanic.

### 🌟 Core Features

    Dynamic Decay System: Perishable items such as vegetables, fruits, fish, eggs, and dairy now have a limited shelf life.

    Environmental Impact:

        Seasonal Effects: Decay accelerates during the Summer and slows down in outdoor environments during the Winter.

        Special Categories: Fish decay 50% faster when kept outside of cold storage.

    Waste-to-Treasure (Recycling Machine):

        Rotten Plant Processing: The Recycling Machine now supports processing "Rotten Plants."

        50:1 Conversion: Input 50 Rotten Plants to produce 1 Deluxe Speed-Gro.

### 🛠️ Installation Requirements

    Install the latest version of SMAPI.

    Place the extracted folder into your Stardew Valley/Mods directory.

    Note: This mod is fully self-contained; Content Patcher is not required.
